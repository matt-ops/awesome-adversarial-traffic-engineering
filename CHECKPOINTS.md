# Checkpoints

These are optional exit ramps in one course. They add no lessons, gates, files, or assignments.

## Foundation: about 24 focused hours

Complete **Foundation → Learn, Lab, and Self-assess** in all nine modules:

| Module | Time | Start |
|---:|---:|---|
| 0 | 1.5 h | [Safety Foundation](COURSE.md#module-0-foundation) |
| 1 | 3 h | [Request path Foundation](COURSE.md#module-1-foundation) |
| 2 | 2 h | [Automated abuse Foundation](COURSE.md#module-2-foundation) |
| 3 | 4 h | [Browser automation Foundation](COURSE.md#module-3-foundation) |
| 4 | 3 h | [Bot detection Foundation](COURSE.md#module-4-foundation) |
| 5 | 3 h | [Edge and DDoS Foundation](COURSE.md#module-5-foundation) |
| 6 | 5 h | [Python and review Foundation](COURSE.md#module-6-foundation) |
| 7 | 1.5 h | [Experiment and reporting Foundation](COURSE.md#module-7-foundation) |
| 8 | 1 h | [Communication Foundation](COURSE.md#module-8-foundation) |
| **Total** | **24 h** | |

At this point you can explain the field and demonstrate each minimum skill once. Stop here or continue with Applied.

## Applied: about 7 days

Complete **Applied → Learn, Lab, and Self-assess** in Modules 0–8. You can now perform each core skill independently in a guided environment.

## Integrated: about 21 days

Complete **Integrated → Learn, Lab, and Self-assess** in Modules 0–8. You can now connect scope, abuse simulation, browser/agent populations, detection, resilience, reporting, and communication in one capstone.

## Deep: about 6 weeks

Complete **Deep → Learn, Lab, and Self-assess** in Modules 0–8. You can now investigate a narrow question independently, reproduce it safely, evaluate alternatives, and teach the result.

These descriptions report course depth, not a claim of production authorization.
