# Roadmap

The canonical curriculum architecture is fixed. Future work deepens the same nine modules; it must not introduce parallel courses.

## Near term

- Expand concept guides and exact-command walkthroughs.
- Complete all Python and secure code-review exercises.
- Add browser, protocol, and bounded-load lab populations.
- Generate deterministic analysis fixtures and a synthetic capstone report.

## Later

- Add optional Prometheus and Grafana observability.
- Add deeper browser-version drift fixtures.
- Add HTTP/3 and QUIC observation exercises where the local toolchain supports them.
- Add portfolio publishing guidance after privacy and safety review.
