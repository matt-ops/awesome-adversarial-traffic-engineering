# Detection science

A confusion matrix separates true positives, false positives, true negatives, and false negatives. Precision asks what fraction of positive decisions were correct; recall asks what fraction of known positives were found. False-positive rate uses legitimate events as its denominator. Base rates and population mix materially change operational results.

Evaluate by population, not only in aggregate. Start in shadow mode, attach reason codes, compare thresholds and actions, monitor drift and customer harm, and define rollback and retest. A toy score demonstrates method, not a production threshold.
