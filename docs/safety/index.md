# Safety boundary

Course exercises are allowed only against:

- the bundled loopback lab;
- a non-routable isolated target you own;
- the exact target assigned by a training provider; or
- an organization-owned test environment covered by separate written authorization.

Never use real stolen data, unassigned infrastructure, public proxies, malware,
botnets, or routed denial-of-service traffic. Never disable an allowlist, hard
cap, timeout, abort condition, or cleanup requirement.

Read the [local target policy](local-target-policy.md) and [traffic
guardrails](traffic-guardrails.md) before any repository lab.
