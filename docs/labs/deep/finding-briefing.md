# Finding, briefing, and mock lab

Use only synthetic/public-safe course evidence.

## Finding structure

1. Boundary and precondition
2. Blocked or unaffected baseline
3. Controlled adversary action
4. Protected action or service effect
5. Repeatability, negative control, and residual anomalies
6. Root-cause evidence versus hypothesis
7. Limitation and alternate explanation
8. Remediation invariant and exact retest

The complete example is `lab/reports/synthetic-finding.md` in the repository.

## Five-minute briefing card

| Time | Required content |
|---:|---|
| 0:00-0:30 | scope and objective |
| 0:30-1:15 | baseline and attack path |
| 1:15-2:30 | protected effect and evidence |
| 2:30-3:30 | repeatability, anomalies, limits |
| 3:30-4:20 | remediation invariant and retest |
| 4:20-5:00 | decision requested and questions |

## Mock scorecard

Score each dimension 0 (absent), 1 (partial), or 2 (clear and evidence-based):

- correct foundation
- explicit assumptions and authorization
- immediate control-specific recon
- adversary objective and protected effect
- experiment variables and safe boundary
- evidence and alternate explanations
- remediation and exact retest
- public-safe communication

A low score is a repair assignment, not a cue to memorize a phrase. Name the
exact lesson, source section, lab action, and artifact that closes the gap.
