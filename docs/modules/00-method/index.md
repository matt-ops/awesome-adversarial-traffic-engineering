# Module 00 — Method and authorization

**Outcome:** design an authorized, falsifiable engagement whose evidence proves
an adversarial objective rather than merely showing tool output.

| Depth | Complete | Evidence |
|---|---|---|
| Foundation | Lessons 1–3 and their core exercises | Role comparison, engagement plan, experiment plan |
| Applied | Execute the safe-client dry run and rejection test in Lesson 2 | Safety-control output and explanation |
| Integrated | Turn one existing lab weakness into the complete fifteen-step plan in Lesson 3 | End-to-end runbook |
| Deep | Add change control, production evidence gates, and exact retest ownership | Reviewable engagement appendix |
