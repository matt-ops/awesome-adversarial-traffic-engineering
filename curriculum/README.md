# Curriculum

AATE has one canonical learning path. The [path](path.md) sequences nine modules through four cumulative depths:

1. Foundation → 24 cumulative focused-hour exit ramp
2. Applied → 7-day exit ramp
3. Integrated → 21-day exit ramp
4. Deep → 6-week exit ramp

Checkpoint files are concise completion views. They link to exact module sections, collect required artifacts, and point to a readiness gate. All teaching, labs, resource assignments, misconceptions, and production limitations live in the module pages.
