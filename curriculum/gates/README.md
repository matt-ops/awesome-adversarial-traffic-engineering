# Readiness gates

Every checkpoint evaluates the same five dimensions at increasing depth:

1. **Explain:** describe mechanisms without substituting product names for understanding.
2. **Build or run:** produce the required local evidence.
3. **Measure:** interpret quantitative results and uncertainty.
4. **Communicate:** present a concise finding, design, or interview answer.
5. **Operate safely:** state authorization, scope, caps, abort conditions, evidence handling, limitations, and retest.

A gate passes only when all five dimensions pass and all nine modules have reached the corresponding level. The result describes demonstrated curriculum evidence; it is not a hiring prediction.
