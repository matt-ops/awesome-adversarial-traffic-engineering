# Integrated gate

Prerequisite: every module is at level 3 and the integrated capstone evidence chain is complete.

| Dimension | Pass evidence |
|---|---|
| Explain | Defend cross-layer design choices, alternative explanations, and control tradeoffs under follow-up. |
| Build or run | Reproduce the complete synthetic engagement with four client populations and layered controls. |
| Measure | Present per-population error, feature ablation, service impact, and recovery evidence. |
| Communicate | Deliver the technical report, executive summary, briefing, and full mock loop. |
| Operate safely | Show the decision log, monitoring, abort behavior, recovery validation, evidence handling, and scoped retest. |

Pass label: **Loop-ready with a credible hands-on portfolio.**
