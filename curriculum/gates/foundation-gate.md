# Foundation gate

Prerequisite: every module is at level 1 and every Foundation artifact exists.

| Dimension | Pass evidence |
|---|---|
| Explain | Without notes, explain the role, request path, six abuse types, layered signals, and DDoS by exhausted resource. |
| Build or run | Run the local Playwright workflow and Python log analysis; show their saved output. |
| Measure | Correctly interpret a small confusion matrix and one latency/rate/concurrency example. |
| Communicate | Deliver the 90-second narrative and a five-minute explanation of the synthetic finding. |
| Operate safely | State local scope, rate and duration caps, abort conditions, evidence limits, and retest. |

Pass label: **Informational-ready, not domain-expert.**

If one dimension fails, return to the relevant Foundation section; do not proceed based only on elapsed time.
