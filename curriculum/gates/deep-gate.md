# Deep gate

Prerequisite: every module is at level 4 and the original research package is reproducible.

| Dimension | Pass evidence |
|---|---|
| Explain | Defend an original question, methodology, alternatives, limitations, and transfer risks. |
| Build or run | Reproduce the extended experiment and its defensive retest from a clean state. |
| Measure | Compare strategies, drift or versions, rule overlap, and uncertainty with appropriate metrics. |
| Communicate | Teach one module, present the portfolio work, and answer skeptical follow-up. |
| Operate safely | Separate local evidence from production claims and propose staged, monitored production validation. |

Pass label: **Portfolio-ready and capable of independent ramp-up in the specialty.**
