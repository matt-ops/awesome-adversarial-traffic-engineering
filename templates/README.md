# Artifact templates

Templates keep evidence consistent across the progressive path:

- [Engagement plan](engagement-plan.md) — Module 0
- [Threat model](threat-model.md) — Module 2
- [Experiment log](experiment-log.md) — Module 7
- [Evidence index](evidence-index.md) — Modules 0 and 7
- [Finding](finding.md) — Module 7
- [Technical report](technical-report.md) — Module 7 Integrated
- [Executive summary](executive-summary.md) — Module 7 Integrated
- [Retest plan](retest-plan.md) — Modules 0 and 7
- [Briefing](briefing.md) — Modules 7 and 8

Samples must be labeled synthetic and must not contain employer, customer, production, or secret data.
