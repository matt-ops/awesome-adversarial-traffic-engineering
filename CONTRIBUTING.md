# Contributing

Contributions must preserve one path, nine modules, and four cumulative depth levels. Add a topic to the relevant module rather than creating a new course or track.

Resource additions need a depth label, priority label, module, source type, concise annotation, and “why it matters” statement. Prefer primary sources.

Lab changes must remain local-only, synthetic, deterministic where practical, conservatively capped, and covered by safety tests. Do not add remote-target overrides, secrets, production data, or offensive integrations prohibited by [SAFETY.md](SAFETY.md).

Before review, run the curriculum validator and available tests, check relative links, and explain any environment-limited validation in `STATUS.md`.
