# Awesome adversarial traffic engineering resources

This catalog is selective rather than exhaustive. Every entry has exactly one depth label, one priority label, a canonical module, a source type, an annotation, and a reason it matters.

Depth labels:

- `[L1 Foundation]`: minimum role literacy
- `[L2 Applied]`: guided independent demonstration
- `[L3 Integrated]`: end-to-end analysis and design
- `[L4 Deep]`: implementation detail or original research

Priority labels:

- `[Required]`: required at the labeled checkpoint
- `[Recommended]`: high-value reinforcement
- `[Optional]`: select for a question or gap

The Foundation checkpoint uses only the small Required subset named in module Foundation sections. A Level 1 label does not mean every Level 1 catalog entry is required.
