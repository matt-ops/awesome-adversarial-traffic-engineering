# AI browser agents

Agent frameworks are not required for Foundation or Applied work. Use them only against owned local environments and distinguish model policy, framework behavior, browser behavior, and task design.

- [Playwright MCP](https://github.com/microsoft/playwright-mcp) — `[L3 Integrated]` `[Recommended]` · Modules 2–4 · Project repository. Browser-control tools built on Playwright. It matters for threat modeling agent tool interfaces without targeting third parties.
- [BrowserGym](https://github.com/ServiceNow/BrowserGym) — `[L3 Integrated]` `[Recommended]` · Modules 2–4 · Research framework. Environments and evaluation for web agents. It matters for separating task benchmarks from detection claims.
- [WebArena](https://github.com/web-arena-x/webarena) — `[L4 Deep]` `[Optional]` · Modules 2–4 · Research framework. Realistic web-agent benchmark environment. It matters for deeper agent behavior research, not first-checkpoint study.
- [browser-use](https://github.com/browser-use/browser-use) — `[L4 Deep]` `[Optional]` · Modules 2–4 · Project repository. An agentic browser framework. It matters as one implementation to compare; do not treat framework artifacts as universal agent signals.
