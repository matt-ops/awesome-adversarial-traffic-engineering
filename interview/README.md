# Interview preparation

Interview preparation is Module 8 of the canonical path, not an unrelated course. Answers should cite evidence produced in Modules 0–7.

| Interview evidence | Source module/artifact |
|---|---|
| Safe engagement answer | Module 0 checklist, plan, and decision log |
| Request-path design | Module 1 architecture and telemetry map |
| Threat model | Module 2 abuse map |
| Browser deep dive | Module 3 Playwright/CDP evidence |
| Detection system | Module 4 evaluation and ablation |
| DDoS/edge design | Module 5 bounded resilience report |
| Coding/review | Module 6 tested solutions and review packages |
| Technical finding | Module 7 report, remediation, and retest |
