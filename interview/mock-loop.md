# Five-session mock loop

Run the loop only after the Integrated sections are complete.

1. **Practical Python:** 60 minutes; parse, group, bound work, test, and explain.
2. **Secure code review:** 60 minutes; identify behavior, preconditions, impact, fixes, tests, and telemetry.
3. **Bot and browser domain:** 60 minutes; automation, CDP, signals, state, metrics, and false positives.
4. **DDoS and edge design:** 60 minutes; architecture, exhausted resources, safe test, controls, and recovery.
5. **Behavioral and leadership:** 60 minutes; two stories with deep follow-up plus role narrative.

After each session, complete the [scorecard](scorecards.md), cite the supporting module artifact, and record one correction to facts, one correction to structure, and one next practice task.
