# Security policy

Report a target-validation bypass, unsafe load ceiling, secret, privacy issue, or other security defect privately to the repository owner before public discussion. Include the affected file, a minimal local reproduction, impact, and a proposed guardrail when possible.

Do not validate a suspected issue against any external service. Keep proof of concept traffic local and bounded.

This educational repository does not accept reports about bypassing third-party bot, WAF, or DDoS products.
